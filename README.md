# 智能导览

#### 基本思路

a级客户端通过arduino对应传感器识别是否有生物体接近，如有调用人脸识别，连接服务端获取所关联的用户位置并显示

b级客户端通过arduino对应传感器识别是否有生物体接近，如有调用人脸识别，连接服务端上传对应用户的位置

服务端维护数据库与人脸数据
